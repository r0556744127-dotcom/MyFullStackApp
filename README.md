## 📖 על הפרויקט

**MyFullStackApp** היא פלטפורמת לימוד וירטואלית שפותחה במיוחד עבור מצבי חירום, המאפשרת למוסדות חינוך להמשיך בפעילות לימודית מלאה מרחוק.

המערכת מספקת סביבה מאובטחת ונוחה לתלמידים, מורים וצוות בית הספר — כולל ניהול שיעורים, מטלות, הגשות וציונים.

## 🚀 הרצה מקומית

### דרישות מוקדמות
- [.NET 8 SDK](https://dotnet.microsoft.com/download)
- [Node.js 18+](https://nodejs.org/)
- SQL Server

### 1️⃣ Backend
```bash
cd RachelProject/WebApplication
dotnet restore
dotnet run
```

### 2️⃣ Frontend
```bash
cd ריאקט
npm install
npm run dev
```

### 3️⃣ פתח בדפדפן
```
http://localhost:5173
```

## 📁 מבנה הפרויקט

```
MyFullStackApp/
├── RachelProject/
│   ├── WebApplication/      ← ASP.NET Core API + Controllers
│   ├── Repositories/        ← Data Access Layer
│   ├── service/             ← Business Logic Layer
│   └── codeFirst/           ← EF Core Migrations & Models
└── ריאקט/                   ← React Frontend
    └── src/
        ├── components/      ← רכיבים משותפים
        ├── pages/           ← דפי האפליקציה
        ├── services/        ← קריאות API
        └── features/        ← Redux slices
```
