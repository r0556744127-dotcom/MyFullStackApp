## 💡 הרעיון מאחורי הפרויקט

בשנים האחרונות, מצבי חירום לימדו אותנו שהחיים אינם עוצרים — והלימודים אינם צריכים לעצור אף הם.

תלמידים רבים כבר הורגלו ללמוד מרחוק, להשתמש בכלים כמו זום ובמרחבים וירטואליים שונים. אולם הכלים הקיימים אינם תמיד עונים על מכלול הצרכים של מוסד חינוכי שלם — ניהול כיתות, מעקב אחר מטלות, הגשות, ציונים ותקשורת רציפה בין מורים לתלמידים.

.**MyFullStackApp** נולדה מתוך צורך זה — מערכת אחת, מקיפה ומשולבת, המאפשרת לבית הספר לתפקד במלואו, אף מרחוק

---

## 🎯 מה המערכת מציעה?

המערכת מספקת ממשק נוח ומאובטח לשלושה סוגי משתמשים:

- 👨‍🎓 **תלמידים** — כניסה אישית, צפייה בשיעורים, הגשת מטלות ומעקב אחר ציונים
- 👩‍🏫 **מורים** — ניהול כיתות משויכות, העלאת שיעורים, מטלות ומתן ציונים
- 🏫 **מנהל** — שליטה מלאה על מבנה בית הספר, תלמידים, מורים וכיתות

## 🚀 הרצה מקומית

### דרישות מוקדמות
- [.NET 8 SDK](https://dotnet.microsoft.com/download)
- [Node.js 18+](https://nodejs.org/)
- SQL Server

### 1️⃣ Backend
```bash
cd RachelProject/WebApplication
dotnet restore
dotnet run
```

### 2️⃣ Frontend
```bash
cd ריאקט
npm install
npm run dev
```

### 3️⃣ פתח בדפדפן
```
http://localhost:5173
```

## 📁 מבנה הפרויקט

```
MyFullStackApp/
├── RachelProject/
│   ├── WebApplication/      ← ASP.NET Core API + Controllers
│   ├── Repositories/        ← Data Access Layer
│   ├── service/             ← Business Logic Layer
│   └── codeFirst/           ← EF Core Migrations & Models
└── ריאקט/                   ← React Frontend
    └── src/
        ├── components/      ← רכיבים משותפים
        ├── pages/           ← דפי האפליקציה
        ├── services/        ← קריאות API
        └── features/        ← Redux slices
```
